Web Fonts

A short layout by Sarah Hoare

Working with fonts on the web can feel like crossing a RAGING river filled with all sorts of dangerous things, like browser compatibility issues, bad rendering, and even security problems you'll need to overcome. But if you're brave enough to cross this river, you'll be rewarded with beautiful, unique work that stands out.

The CSS3 @font-face rule allows web developers to specify custom fonts for their web pages. By allowing fonts to be downloaded from a server, @font-face eliminates the need to depend on the limited number of fonts users have installed on their computers.

@font is a powerful tool that gives web designers a greater level of control over the appearance of their sites. It opens a whole new world of possibilities for how web pages can look, making them more expressive and unique.

A few of the most used properties include:
  • font-family: specifies the font family to use.
  • font-size: specifies the size of the font.
  • font-weight: specifies the weight, or boldness, of the font.
  • font-style: specifies whether the font should be italicized.

Requirements:

  • Link to Roboto from Google Fonts
  • Match this style as close as possible
  • Submit the html and CSS within a zip file

Disclaimer:

Some properties can be specified in shorthand, using the font property. You should reuse your CSS whenever possible. Assume, someone will need to build additional pages using your work.
